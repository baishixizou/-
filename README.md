# 初中数学练习课堂

一个无需后端的课堂练习 Web App，面向学生练习计算，面向老师查看练习情况。当前覆盖：

- 一元一次方程
- 二元一次方程组
- 一元一次不等式

## 功能

- 学生端：添加/选择学生、切换题型、自动出题、提交答案、即时反馈与提示。
- 教师端：按学生展示完成数量、答对数量、正确率、分题型正确率和最近练习记录。
- 数据管理：练习数据保存在当前浏览器 `localStorage`，支持导出 JSON 和清空数据。

## 本地运行

```bash
npm start
```

然后访问 <http://127.0.0.1:4173>。

## 测试
